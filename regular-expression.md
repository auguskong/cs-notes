## 特殊字符

`^`: 表示行首

`$`: 行尾

`* `: match 0 or more of the preceding token

`?`: match between 0 and 1 of the preceding token

`+`: match 1 or more of the preceding token

`\d`: match any digit character (0 - 9)



`py` 和 `^py` 的区别: 

* `py`可以匹配行中的 比如jkdepy
* `^py` 只能匹配 以 `py` 开头的 比如pyjde 而不能匹配 dejkpy
